ERROR: 'directory' not specified in url parameters.

To fix this, you need to edit your IGV settings:
1) go to View > Preferences...
2) select the 'Advanced' tab
2) check the "Edit server properties" checkbox
3) copy-paste this url in the Data Registry URL text box:

http://maclab-utils:8000/scripts/dataServerRegistry.py?genome=$$&directory=<your igv files directory on the cluster>

For example:

http://maclab-utils:8000/scripts/dataServerRegistry.py?genome=$$&directory=/humgen/atgu1/fs03/weisburd/igv_files




NOTE: after creating that directory, make sure it can be accessed by this server.

To do that, run this command on the cluster: chmod 777 <your igv files directory on the cluster>
